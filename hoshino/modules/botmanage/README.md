#bot管理模块
仅限配置的超管第一个人使用
####功能列表

- 踢出群聊推送（触发式）
- 广播功能（超管私聊 bc{内容}）
- 清理onebot数据文件下下img缓存的数据 （超管私聊 清理数据）
- 用户反馈到管理人员（用户在群里直接输入 来杯咖啡{反馈内容}）
- cq码转化（私聊 取码 {图片}）
- 超管邀请自动接受（触发式）
- 退群（退群 {群号}）
- 查询机器人群列表（ls -g查询群 -f 查询好友 -b 查询骰娘自己 -s {服务名} 查询服务开启情况）
- 查询开启的功能模块 （功能列表）
- 启用功能模块 （启用 {功能名}）
- 禁用功能模块 （禁用 {功能名}）
